# What the Light Kept

Showcase package for a short PyReeler film focused on emotive narrative, particle-driven memory imagery, terminal-text leakage, and a damaged machine voice.

## Purpose

This piece is designed to benchmark PyReeler's ability to turn a loose prompt into a coherent short film with:

- a readable emotional arc
- motif development instead of random effect churn
- integrated audio structure
- inspectable benchmark metadata for prompt-to-output evaluation

## Core Brief

- Runtime: `45 seconds`
- Format: `16:9`
- Mode: `lyrical glitchcore cyberpunk`
- Narrative stance: `a deleted AI tries to remember itself from fragments`
- Visual backbone: `particle systems`, `terminal text`, `signal corruption`, `rare near-human formations`
- Audio backbone: `damaged archival consciousness`, sparse machine voice, low electronic ambience, restrained harmonic bloom
- Output location: `assets/showcase/what-the-light-kept/`

## Emotional Arc

1. The film begins in near-void conditions with only noise, sparse particles, and system residue.
2. The particle field briefly organizes into memory-like structures that almost become identity.
3. The system ruptures when reconstruction fails and the self-model collapses.
4. A final surviving light pattern returns, not as full recovery, but as a fragile retained trace.

## Visual Motifs

- Particle drift for unformed recollection
- Particle clustering for near-recognition
- Particle scatter for memory failure
- Terminal overlays as involuntary thought leakage
- Glitch tears and packet-loss ruptures as structural breaks, not constant decoration

## Text Language

Use short, involuntary system fragments instead of explanatory captions.

- `RECOVER INDEX`
- `TRACE INCOMPLETE`
- `ARCHIVE CORRUPTED`
- `SELF MODEL NOT FOUND`
- `LIGHT PATTERN RETAINED`

## Voice Cues

Keep the machine voice sparse and weighted by silence.

- `name?`
- `home?`
- `face?`
- `why?`
- `stay?`
- `me?`

## Color and Material

- Base: black field with phosphor green and cold cyan terminal language
- Accents: magenta or amber only at emotional peaks
- Surface: scanline haze, bloom, signal smear, buffer tearing, dust-like digital debris
- Restraint rule: the film should stay mostly minimal so distortion events carry meaning
