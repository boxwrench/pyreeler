# Scratch Notes

This directory is for loose observations, half-ideas, and unstructured experimentation logs. No format required. No pressure to be coherent.

## Naming

`YYYY-MM-DD-brief-description.md`

Or don't name them well. It's fine.

## Content Ideas

- Random parameter combinations tried
- "What if I..." questions
- Links to inspiration (demoscene, shader toy, etc.)
- Failed experiments and post-mortems
- Performance observations
- Half-baked theories
